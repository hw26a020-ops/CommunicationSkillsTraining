# 説明力向上ゲーム「相手に伝わる説明」仕様書

## 1. ゲームコンセプト
「相手に伝わる説明」ができるようになることを目的とした、説明力トレーニングゲーム。

## 2. ゲームフロー
1. **タイトル画面**
   - ゲーム起動時に表示される。
   - ゲームの概要やルールが説明されており、「スタート」ボタンを押すことでゲームが開始される。
2. **難易度の選択**
   - プレイヤーはゲーム開始時に難易度（初級・中級・上級・チャレンジャー）を選択する。
3. **お題の提示**
   - AIが、選択された難易度に応じた適切なお題をその場で新しく生成し、プレイヤーに提示する。
4. **説明の入力**
   - プレイヤーは難易度ごとに設定された制限時間内に、お題に対する説明をテキストで入力する（テキスト入力のみ）。
   - 中級以上の難易度では、設定された最低文字数を満たす必要がある。
   - 制限時間が0になった場合、その時点で入力されているテキストを自動送信して採点フェーズに移行する。
5. **AIによる判定・評価**
   - AIが「初めてそのお題を聞く人」というペルソナとして、プレイヤーの説明を理解できるか判定する。
6. **スコア獲得・結果表示**
   - AIが評価基準を総合的に判断し、全体の総合点（100点満点）を算出する。
   - 採点結果とともに、プレイヤーの文章に対する「良かった点」と「改善点」を具体的にフィードバックする。

## 3. 難易度設定、制限時間、および最低文字数
難易度によって「AIがその場で生成するお題の難しさ」「制限時間」「最低文字数」「AIの採点の厳しさ」が変化する。

- **初級**
  - **お題の難しさ**：日常的で直感的に理解・説明しやすいお題（例：信号機、トランプなど）。
  - **制限時間**：180秒（3分）
  - **最低文字数**：なし
  - **AIの採点基準**：非常に甘い。多少の言葉足らずや論理の飛躍があっても、AIが前後の文脈から好意的に解釈して高得点をつけやすい。
- **中級**
  - **お題の難しさ**：日常的に使うが、仕組みの理解が必要なお題（例：電子レンジ、エレベーターなど）。
  - **制限時間**：240秒（4分）
  - **最低文字数**：100文字以上
  - **AIの採点基準**：標準的。一般的な大人が理解できるレベルの客観性とわかりやすさを基準に採点する。
- **上級**
  - **お題の難しさ**：抽象的、または科学的な理解・説明が必要なお題（例：Wi-Fi、重力など）。
  - **制限時間**：300秒（5分）
  - **最低文字数**：200文字以上
  - **AIの採点基準**：厳しい。論理的な矛盾や説明不足、曖昧な表現を厳しくチェックし、減点対象とする。
- **チャレンジャー**
  - **お題の難しさ**：定義や仕組みが極めて複雑で、専門知識を必要とするお題（例：量子力学、ブロックチェーンなど）。
  - **制限時間**：360秒（6分）
  - **最低文字数**：300文字以上
  - **AIの採点基準**：極めて厳しい。専門用語を一般的な分かりやすい言葉に徹底的に翻訳できているか、構成に一切の妥協がないかを厳密に審査する。最高の基準を満たさなければ高得点は獲得できない。

## 4. AI評価基準とフィードバック内容
AIは以下の6つの観点を総合的に評価し、全体の総合点（100点満点）を算出、およびフィードバックを生成する。

### 評価の観点
- **正確性**：説明内容が事実として正しいか。
- **わかりやすさ**：表現が平易で、直感的に理解しやすいか。
- **論理性**：説明の筋道が通っているか。
- **専門用語の少なさ**：難しい言葉や専門用語を避けているか（あるいは適切に補足されているか）。
- **具体例の有無**：理解を助ける具体例が提示されているか。
- **文章構成**：導入・本論・結論などの構成が整っているか。

### フィードバック内容
- **全体の総合点**：100点満点のスコア
- **良かった点**：説明の中で特に伝わりやすかった部分や, 工夫が感じられた点を具体的に評価する。
- **改善点**：より伝わる説明にするために, 不足している情報や書き直すべき表現を具体的にアドバイスする。

## 5. 画面構成とレイアウト設計

### 5.1. タイトル画面
- **メインタイトル**
  - 「相手に伝わる説明力トレーニング」
- **サブタイトル**
  - 「AIが判定する、あなたの『説明する力』」
- **イントロダクション**
  - 簡単な3ステップ解説（1. 難易度選択 -> 2. 時間内テキスト入力 -> 3. AIによる採点とフィードバック）
- **操作部**
  - 「ゲームを始める」ボタン（クリックで難易度選択へ遷移）

### 5.2. 難易度選択画面
- **ヘッダー**
  - 「挑戦する難易度を選択してください」のガイドライン
- **難易度カード（4つの選択肢）**
  - **初級**：日常的で説明しやすいお題（制限時間: 180秒、最低文字数: なし、採点: 甘口）
  - **中級**：仕組みの理解が必要なお題（制限時間: 240秒、最低文字数: 100文字、採点: 普通）
  - **上級**：抽象的・科学的なお題（制限時間: 300秒、最低文字数: 200文字、採点: 辛口）
  - **チャレンジャー**：定義や仕組みが極めて複雑なお題（制限時間: 360秒、最低文字数: 300文字、採点: 極辛）
- **操作部**
  - 「タイトルに戻る」ボタン

### 5.3. ゲームプレイ画面
- **上部ステータスバー**
  - 現在の選択難易度表示
  - 残り時間カウントダウンタイマー（例：03:45）
- **お題提示エリア**
  - AIが生成したお題を大きく中央に表示（例：「今日のお題：『電子レンジ』」）
  - **※初級・中級・上級のみ補足説明を表示。チャレンジャーでは補足説明を非表示にする。**
- **テキスト入力エリア**
  - 説明文を入力する複数行テキストエリア。
  - 現在の入力文字数 / 必要最低文字数のインジケーター（例：「120 / 100文字」）
- **フッター操作部**
  - 「送信して採点する」ボタン（最低文字数未達の場合は無効化、または警告表示）
  - 「あきらめる」ボタン
    - **※「あきらめる」ボタンを押すと確認ダイアログが表示され、そこまで書いた内容で即座に採点される。**

### 5.4. 判定ローディング画面
- **進捗表示**
  - 「AIがあなたの説明を読み込んでいます...」
  - 「初めてお題を聞く人の視点で理解度を検証中...」などの読み込みメッセージを表示。

### 5.5. 結果表示画面
- **総合スコア**
  - 100点満点中の獲得点数をグラフィカルに表示。
- **フィードバックエリア**
  - 「良かった点」（ポジティブ評価）をわかりやすく配置。
  - 「改善点」（具体的なアドバイス）をわかりやすく配置。
- **操作部**
  - 「もう一度遊ぶ」ボタン（同じ難易度でお題を再生成）
  - 「難易度選択に戻る」ボタン（難易度選択画面へ）
  - 「タイトルに戻る」ボタン（タイトル画面へ）
